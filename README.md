# MyFirstWebApp

This is a simple project to test my ability in Python for the back-end and try to use html, css and JS for the front-end. 

## App Purpose

The app aims to display the effect of rebalancing frequency and transaction costs on portoflio performance. Usually, building a portfolio entails the development of a strategy to determine the portofolio composition. In order to stand by the chosen strategy, it is necessary to rebalance portfolio, i.e. to adjust the exposure towards each asset to match the intended strategy.
